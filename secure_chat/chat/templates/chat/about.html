About Secure Chat


Secure Chat is a college project built by final-year students aiming to create a chat application with a focus on security and privacy. This project is developed to cater to individuals and organizations who prioritize secure communication without leaving digital footprints.



Why was this project built?


As technology advances, big-tech companies collect more and more personal data about us. This is concerning for individuals and organizations that want to securely share sensitive information without the fear of it being tracked or misused. Our project uses Dual encryption (N-TEA and AES) to ensure that communication remains secure, private, and without any digital footprints. We, as engineering students, wanted to contribute to society by addressing these privacy concerns. Adding even a small contribution to secure communication is meaningful to us.



How and When was this project built?


This project was built as part of our final year college project. The main goal was to design a secure chat application that uses advanced encryption techniques to safeguard messages from any unauthorized access. We started working on it in our final year, combining our knowledge of encryption algorithms and real-time web technologies to create a solution that aligns with our vision for a more private and secure digital world.
